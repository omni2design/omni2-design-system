# Omni2 Design System

A scalable design system powering Omni2 Design products, client platforms, and AI-assisted development workflows.

## Overview

This system connects:
- Figma (design)
- Tailwind (implementation)
- Claude & Cursor (AI-assisted generation)

into a unified design language.

## Structure

### Base Tokens
Core primitives:
- Color
- Typography
- Spacing
- Radius

### Semantic Tokens
Purpose-based design decisions:
- Background
- Text
- Border
- Button states

### Themes
Project-specific adaptations:
- Omni2 (default brand)
- ArchRecon (AEC platform)
- AMC (contractor / homeowner workflows)

### Tailwind
Production-ready mapping of tokens into frontend code.

### AI Skills
Guidelines for ensuring consistent system usage across AI-generated UI.

## Philosophy

Do not create a new design system for each project.

Instead:
- Build one core system
- Extend with themes
- Add product-specific components only when necessary

## Outcome

This approach enables:
- Faster product development
- Consistent user experience
- Seamless design-to-code workflows
- AI-assisted UI generation grounded in real systems
