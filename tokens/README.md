# Omni2 Tokens

This folder contains the source token structure for Omni2 Design System.

## Token layers

1. Base tokens  
Raw design values such as color, spacing, radius, and typography.

2. Semantic tokens  
Purpose-based usage such as background, text, border, and button roles.

3. Theme tokens  
Project-specific mappings for Omni2, AMC, and ArchRecon.

## Rule

Use semantic tokens first. Use theme tokens for project-specific branding. Avoid raw hex values in product code unless creating a new token.
